# Collect Micro Journal

Date: 2026-07-20

Tiny app concept: a soft arcade **micro journal**.

Core touches:

- random prompt generation
- one-click reset
- daily streak tracking

Palette:

- Background: `#1b1b3a`
- Accent: `#693668`
- Text: `#a74482`

Starter file:

```html
<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Collect Micro Journal</title>
  <style>
    body {
      margin: 0;
      min-height: 100vh;
      display: grid;
      place-items: center;
      background: #1b1b3a;
      color: #a74482;
      font-family: system-ui, sans-serif;
    }

      width: min(34rem, calc(100% - 2rem));

    }

      border: 0;
      border-radius: 6px;

    }
  </style>
</head>
<body>
  <main>
    <h1>Collect Micro Journal</h1>
    <p>A soft arcade micro journal with random prompt generation, one-click reset, and daily streak tracking.</p>
    <button id="spark">Make today count</button>
    <p id="out"></p>
  </main>
  <script>
    const notes = ["Start tiny.", "Ship the useful bit.", "Make it pleasant."];
    spark.onclick = () => out.textContent = notes[Math.floor(Math.random() * notes.length)];
  </script>
</body>
</html>
```
